## `Задачи`

`Телеграмм-бот`

Телеграмм-бот разработан для орнитологов-любителей и иных лиц, интересующихся птицами.  
Бот принимает фотографию, находит птиц на фотографии и предоставляет информацию об обнаруженных птицах.
